## 關於 HugoWind

HugoWind 是一個免費、可自訂且可用於生產環境的 Hugo + Tailwind CSS 模板。

### 我們的使命

我們致力於為開發者提供一個現代、快速且美觀的網站模板，結合 Hugo 靜態網站生成器的強大功能和 Tailwind CSS 的靈活性。

### 功能特性

- 🚀 快速且效能優化
- 🎨 受 AstroWind 啟發的精美設計
- 🌐 完整的多語言支援（英文、簡體中文、繁體中文）
- 🌙 深色模式支援
- 📱 完全響應式設計
- 🔍 內建搜尋功能
- 📊 SEO 優化
