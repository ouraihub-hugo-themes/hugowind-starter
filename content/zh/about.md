## 关于 HugoWind

HugoWind 是一个免费、可定制且可用于生产环境的 Hugo + Tailwind CSS 模板。

### 我们的使命

我们致力于为开发者提供一个现代、快速且美观的网站模板，结合 Hugo 静态网站生成器的强大功能和 Tailwind CSS 的灵活性。

### 功能特性

- 🚀 快速且性能优化
- 🎨 受 AstroWind 启发的精美设计
- 🌐 完整的多语言支持（英文、简体中文、繁体中文）
- 🌙 暗色模式支持
- 📱 完全响应式设计
- 🔍 内置搜索功能
- 📊 SEO 优化
